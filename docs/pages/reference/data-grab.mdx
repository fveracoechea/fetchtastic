# DataGrab Class

The `DataGrab` class represents an HTTP request configuration. It provides methods for setting headers, URL parameters, request body, and other options. It also provides convenience methods for performing common HTTP methods such as GET, POST, PUT, DELETE, OPTIONS, PATCH, and HEAD.

## Static Method

### `clone(instance: DataGrab): DataGrab`

Creates a new instance of the `DataGrab` class by cloning an existing instance.

-   `instance`: The `DataGrab` instance to clone.

Returns a new `DataGrab` instance with the same configuration as the cloned instance.

## Properties

-   `url: string`: The URL of the request, including any URL parameters.
-   `searchParams: Record<string, string>`: An object representing the URL parameters of the request.
-   `headers: Record<string, string>`: An object representing the headers of the request.
-   `get: DataGrabResolver<GET>`: A `DataGrabResolver` instance for performing a GET request.
-   `post: DataGrabResolver<POST>`: A `DataGrabResolver` instance for performing a POST request.
-   `put: DataGrabResolver<PUT>`: A `DataGrabResolver` instance for performing a PUT request.
-   `delete: DataGrabResolver<DELETE>`: A `DataGrabResolver` instance for performing a DELETE request.
-   `options: DataGrabResolver<OPTIONS>`: A `DataGrabResolver` instance for performing an OPTIONS request.
-   `patch: DataGrabResolver<PATCH>`: A `DataGrabResolver` instance for performing a PATCH request.
-   `head: DataGrabResolver<HEAD>`: A `DataGrabResolver` instance for performing a HEAD request.

## Constructor

### `constructor(baseUrl?: string | URL, controller?: AbortController)`

Creates a new instance of the `DataGrab` class.

-   `baseUrl: string | URL`: The base URL of the request. Optional.
-   `controller: AbortController`: An `AbortController` instance for aborting the request. Optional.

## Methods

### `setHeaders(data?: HeadersInit, replace = false): DataGrab`

Sets the headers of the request.

-   `data: HeadersInit`: The headers to set.
-   `replace: boolean`: Indicates whether to replace the existing headers or append to them. Default is `false`.

Returns the `DataGrab` instance.

### `appendHeader(name: string, value: string): DataGrab`

Appends a header to the request.

-   `name: string`: The name of the header.
-   `value: string`: The value of the header.

Returns the `DataGrab` instance.

### `deleteHeader(name: string): DataGrab`

Deletes a header from the request.

-   `name: string`: The name of the header to delete.

Returns the `DataGrab` instance.

### `setUrl(url: string | URL, replace = false): DataGrab`

Sets the URL of the request.

-   `url: string | URL`: The URL to set.
-   `replace: boolean`: Indicates whether to replace the existing URL or append to it. Default is `false`.

Returns the `DataGrab` instance.

### `setSearchParams(data?: SearchParamInput, replace = false): DataGrab`

Sets the URL parameters of the request.

-   `data: SearchParamInput`: The URL parameters to set.
-   `replace: boolean`: Indicates whether to replace the existing URL parameters or append to them. Default is `false`.

Returns the `DataGrab` instance.

### `appendSearchParam(name: string, value: string | number | boolean): DataGrab`

Appends a URL parameter to the request.

-   `name: string`: The name of the URL search parameter.

-   `value: string | number | boolean`: The value of the URL search parameter.

Returns the `DataGrab` instance.

### `deleteSearchParam(name: string): DataGrab`

Deletes a URL parameter from the request.

-   `name: string`: The name of the URL parameter to delete.

Returns the `DataGrab` instance.

### `setBody(body: BodyInit | null | unknown): DataGrab`

Sets the body of the request.

-   `body: BodyInit | null | unknown`: The body of the request.

Returns the `DataGrab` instance.

### `setOptions(options: DataGrabOptions, replace = false): DataGrab`

Sets the options of the request.

-   `options: DataGrabOptions`: The options to set.
-   `replace: boolean`: Indicates whether to replace the existing options or merge them with the new options. Default is `false`.

Returns the `DataGrab` instance.

### `getOptions(method: HttpMethod): FetchOptions`

Gets the options object for the request.

-   `method: HttpMethod`: The HTTP method of the request.

Returns the options object for the request.
